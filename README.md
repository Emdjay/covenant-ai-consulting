# Covenant AI Consulting

**We don't just recommend AI. We build it. We proved it on ourselves first.**

An AI workflow consulting business that uses passive screen capture + AI analysis to discover business workflows, then recommends and implements AI-powered automations that measurably increase productivity.

## Repository Structure

| Package | Path | Stack | Purpose |
|---------|------|-------|---------|
| Marketing Site | `marketing-site/` | Next.js, React, TypeScript, Tailwind | Public site: covenantai.consulting |
| Client Portal | `client-portal/` | FastAPI (MVP) → Next.js | Audit results dashboard for clients |
| Capture Agent | `capture-agent/` | Electron, Node.js | Desktop app for passive workflow recording |
| Analysis Engine | `analysis-engine/` | Python, Claude API | AI pipeline for workflow pattern detection |
| Docs | `docs/` | Markdown | Case studies, process docs, templates |

## Service Tiers

1. **AI Website Widget** — Covenant AI chat on client sites ($99-299/mo)
2. **Workflow Audit** — 2-week passive capture + AI analysis ($5K-15K)
3. **AI Implementation** — Build the recommended automations ($10K-50K+)
4. **AI Operations Retainer** — Ongoing optimization + new opportunities ($2K-5K/mo)
