# Marketing Site — covenantai.consulting

Public-facing marketing site for Covenant AI Consulting.

## Stack

- Next.js (App Router)
- React 19 + TypeScript
- Tailwind CSS
- Navy/gold brand palette (inherited from Covenant Sites)

## Pages

- `/` — Homepage (problem, process, proof, CTA)
- `/how-it-works` — Detailed 3-phase breakdown
- `/case-study` — Enoch transformation story
- `/pricing` — Service tiers
- `/about` — Team and story
- `/blog` — AI workflow insights (future)
- `/book` — Discovery call scheduling

## Setup

```bash
npm install
npm run dev
```
