# Client Portal

Dashboard where clients view their workflow audit results, opportunity matrix,
and implementation progress.

## Stack (MVP)

- FastAPI + Jinja2 + Tailwind CSS
- SQLite (engagement data)
- D3.js / React Flow (workflow visualizations)

## Features

- Engagement overview and timeline
- Interactive workflow map
- Time heatmap (where hours go by category)
- Opportunity matrix with ROI projections
- Implementation progress tracker
- Recommendation detail pages
